RequestScopeBean 프로젝트에서는  java방식으로 설정함
RequestScopeBeanXml 프로젝트에서는 xml방식으로 설정함

Bean Injection : 빈 주입

	@Autowired 어노테이션을 사용해서
	Bean을 자동으로 주입 받기
	
	Spring core에 prototype, singleton 이 있고
	Spring MVC에서 request, session, application이 추가됨
	
	Request Scope
	 ㄴ Bean을 정의할 때 request scope로 정의하면
	 	요청(request)이 발생할 때마다 Bean 객체가 생성되어서
	 	자동으로 주입됨
	 	
	 	(생성된) Bean은 요청 발생시 주입만 되고 
	 	request scope에 저장되지는 않음
	 	
	 	xml 방식으로 설정해서 byName 으로 주입받을 때만
	 	request scope에 자동으로 저장됨
	 	
	 	java 방식으로 설정할 때는 @RequestScope 어노테이션을 사용함
	 								ㄴ request scope에 자동으로 저장됨
	 								ㄴ RootAppContext 클래스에서 설정함
	 									
	 	xml 방식으로 Bean을 정의할 때, scope="request"로 설정함
	 	
	 	
	 @Component 어노테이션을 사용해서 Bean을 등록하는 경우,
	 이 Bean 들이 들어있는 package를 scan 해야
	 Bean으로 자동 등록됨
	 	ㄴ ServletAppContext.java에서 설정함
	 	   @ComponentScan("패키지명.클래스")
